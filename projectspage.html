EduGamerz - Gamified Learning

EduGamerz is a gamified learning website dedicated to making learning fun for students. We quiz students with questions from a dataset of questions and implement them into games such as tic tac toe, crossword, and hangman.

Student Verse - Dropout and College

Student Verse is an application designed to give students and teachers information about college and dropout rates. This application uses a logistic regression model to predict based on user data if a student will drop and which college they will get into based on their test scores and extracurriculars.
